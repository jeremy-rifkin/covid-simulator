COVID-19 simulator

This is a basic simulation of the spread of COVID-19. Click the icon to add balls to the simulation. These balls represent people. Use the tool to mark balls as infected. Scroll with the tool to open and close walls. The tool can be used to draw lines on the field - these serve as barriers to the spread of the virus. The tool will add walls to the field. These barriers can be opened or closed increase / decrease the spread of the virus. The download and upload icons can be used to save / load simulations. The icon will reset the simulation.
